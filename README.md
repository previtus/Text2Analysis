# Text2Analysis

## Install 

Prerequisite libraries:

- `pip install gensim pandas spacy pyLDAvis nltk bokeh wordcloud`
- run python script with `import nltk` and `nltk.download('stopwords')`
- `python3 -m spacy download en`
